# 📚 Flashcard Quiz App

## 📋 Overview
A professional flashcard study application built with Flutter for the CodeAlpha Internship program. Create, study, and quiz yourself with custom flashcards to master any subject efficiently.

## ✨ Features

### 📝 Flashcard Management
- **Create Flashcards**: Add new cards with questions and answers
- **Edit Flashcards**: Modify existing cards anytime
- **Delete Flashcards**: Remove unwanted cards
- **View All Cards**: Browse through your complete collection

### 🔄 Card Navigation
- **Next/Previous**: Navigate between cards with smooth transitions
- **Show/Hide Answer**: Toggle answer visibility with animation
- **Card Counter**: See your current position in the deck
- **Swipe Support**: Intuitive navigation

### 📊 Quiz Mode
- **Interactive Quiz**: Test your knowledge with flashcards
- **Score Tracking**: See your score as you progress
- **Correct/Wrong**: Mark answers and get instant feedback
- **Results Screen**: View your performance with percentage
- **Retry Option**: Practice until you master the content

### 💾 Data Management
- **Local Storage**: All flashcards saved locally
- **Auto-Save**: Changes saved automatically
- **Persistent Data**: Cards remain after app restart
- **SQLite Database**: Efficient local storage

### 🎨 User Interface
- **Gradient Colors**: Beautiful blue and purple gradients
- **Smooth Animations**: Card flip and transition effects
- **Responsive Design**: Works on all screen sizes
- **Material Design 3**: Modern Flutter design

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| Flutter | Frontend Framework |
| Provider | State Management |
| SQLite (sqflite) | Local Database |
| Path Provider | File System Access |
| Material Design 3 | UI Components |
| Flutter Launcher Icons | Custom App Icon |

## 📲 Installation

### Prerequisites
- Flutter SDK (3.0.0 or higher)
- Android Studio / VS Code
- Android Emulator / Physical Device / Chrome Browser
